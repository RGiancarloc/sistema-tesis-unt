# Base de Datos

Scripts y configuraciones de base de datos.

## Contenido
- `/schema` - Scripts de creación de tablas y estructura
- `/seed` - Datos iniciales (seed data)
- `/migrations` - Migraciones de base de datos
- `/backups` - Scripts de backup

## Estructura
- `01-creacion-tablas.sql` - Creación de todas las tablas
- `02-triggers-auditoria.sql` - Triggers de auditoría
- `03-indices.sql` - Índices adicionales
- `seed/01-seed-data.sql` - Datos iniciales
