# Kubernetes Configurations

Manifiestos de Kubernetes para despliegue en producción.

## Contenido
- Deployments para cada microservicio
- Services y Ingress
- ConfigMaps y Secrets
- Persistent Volume Claims
- Horizontal Pod Autoscalers

## Estructura
- `/base` - Configuraciones base
- `/overlays/dev` - Configuraciones para desarrollo
- `/overlays/staging` - Configuraciones para staging
- `/overlays/prod` - Configuraciones para producción
