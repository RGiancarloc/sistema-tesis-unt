# UI Components

Componentes de UI compartidos entre aplicaciones.

## Contenido
- Componentes de diseño system
- Componentes de formularios
- Componentes de visualización de datos
- Temas y estilos globales

## Stack
- React
- TypeScript
- TailwindCSS
- Storybook (documentación)

## Uso
Este paquete se importa principalmente en:
- apps/web
- apps/mobile (adaptado para React Native)
